## 算法复杂度

算法复杂度分为时间复杂度(Time)和空间复杂度(Space)。时间复杂度是度量算法执行的时间长短；而空间复杂度是度量算法所需存储空间的大小。

### 分类

按数量级递增排列，常见的时间复杂度有：
- 常数阶O(1)
- 对数阶O(log2n)
- 线性阶O(n)
- 线性对数阶O(nlog2n)
- 平方阶O(n2)
- 立方阶O(n3)
- ...
- k次方阶O(nk)
- 指数阶O(2n)
随着问题规模n的不断增大，上述时间复杂度不断增大，算法的执行效率越低。
